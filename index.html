Capsule

A collaborative web app where users create interactive 3D time capsules that unlock on specific dates. Each capsule acts as a playful container for memories—photos, videos, and notes—revealed in an immersive 3D gallery inspired by Japanese gachapon machines. Built with React Three Fiber, MongoDB, and AWS S3, Capsule combines creative design with real-time collaboration.
